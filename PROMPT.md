# 프롬프트 기록

## 1. 프로젝트 개요 작성

### 요청
프로젝트 개요: StockSignal-Web에 대한 상세 정보를 README.md에 추가

### 내용
- **프로젝트 소개**
  - 프로젝트명: StockSignal-Web (웹 기반 주식 매매 신호 및 대시보드 시스템)
  - 프로젝트 목적: 텔레그램 알림의 휘발성 문제 해결 및 웹에서 체계적인 아카이빙
  - 대상 사용자: 실시간 대응이 어려운 직장인 투자자 및 과거 신호 이력 분석 개인 투자자
  - 기존 방법 대비 개선점: 웹 대시보드를 통한 시각화된 데이터와 종목별 히스토리 조회 기능 제공
  - 기대 효과: 투자 신호 누적 데이터를 통한 전략 검증 및 승률 파악, Spring Boot 활용 실무 경험

- **프로젝트 배경 및 범위**
  - 배경: 파이썬 기반 텔레그램 봇에서 웹 전환
  - In-Scope: 
    - 주식 매수/매도 신호 생성 및 저장
    - 텔레그램 봇을 통한 실시간 알림 전송
    - 신호 이력 리스트 및 모닝 브리핑 조회 페이지
  - Out-of-Scope:
    - OAuth2.0 등 고급 보안 처리
    - 증권사 주문 API 연동
    - 한국어 외 다국어 지원

- **사용 기술 스택**
  - 언어: Java 17, JavaScript, HTML, CSS
  - 프레임워크: Spring Boot
  - 버전 관리: Git & GitHub
