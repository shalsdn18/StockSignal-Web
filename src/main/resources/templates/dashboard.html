REST API

  • POST /api/signals — Create a new signal (JSON body)
  • GET /api/signals — List all signals
  • GET /api/signals/{id} — Get signal by ID
  • GET /api/signals/ticker/{ticker} — Filter by ticker
  • GET /api/signals/type/BUY or SELL — Filter by type

Example payload:

{
  "ticker": "AAPL",
  "signalType": "BUY",
  "price": 182.50,
  "message": "Golden cross detected"
}
